Portfolio Detail

Industry Standard Infra Setup maximised the business value for a Leading Fintech Company
Technogy
Infrastructure Service, Digital Transformation

Business Objective

The Customer Is A Conglomerate And 6 Different Business Units Working Under The Umbrella Of Their Brand. They Had Multiple Websites To Manage Transactions, Handle Customer Care And Build Brand

Challenges

All Portals Code Was Scattered In Different Servers, No Proper Documentation, Permission Level Access Was There, Websites Were Frequently Going Down Which Was Affecting Brand Name. Customer Was Looking IT Infra Consulting Services That Can Help Them Define IT Process, Organise The Company’s Digital Kpi’s, Build The Role Level Access To Server And Software And Help Them To Mitigate Risk

Strategy & Solution

  • We From MCS Did The Initial IT Audit And Presented Vulnerabilities To Customer.

  • Created Version Control To Manage Software Code Using Git

  • Server Level Access Is Disabled For All And Only Project Managers Would Get Access To It

  • All Code Management Is Through Version Control And No Direct Access To Server To Manipulate Code Or Database

  • Created Version Control To Manage Software Code Using Git

  • Set Up IT Process : No Direct Update To Production Environment, Any Feedback Or Fix Should First Get Validated In Test Environment And Post Which Move To Production

Business Objective

Organisation Has Clear Cut IT Road Map, Roles And Responsibilities To Manage Server, Softwares Business Continuity Without Any Downtime Or Malware Attacks Full Control On Kpis
